Every magazine you've read, every corporate annual report, every paperback novel spine — the layout was almost certainly built in InDesign. The 2026 release extends the type engine with AI-assisted variable font interpolation and a new multi-channel publish panel that targets print, interactive PDF and reflowable EPUB from a single source layout.

### Document types and output modes

| Document type | Primary output |
|---|---|
| Book / novel | PDF X-3 · EPUB fixed / reflowable |
| Magazine | PDF print-ready · Interactive PDF |
| Brochure | PDF X-4 · Print on demand |
| Annual report | PDF/A · Interactive online |
| Presentation | PDF / EPUB · Animate CC handoff |
| Digital catalogue | Interactive PDF · HTML5 |

### Typography engine

```
Character: font · size · tracking · kerning · baseline shift · language
Paragraph: alignment · indents · leading · hyphenation dictionary · justification
           drop cap · rules (before/after) · paragraph shading
           GREP style (regex-driven character style application)
Style:     paragraph styles · character styles · object styles · table/cell styles
           Next style chain · Based on · Sequential apply
```

### 2026 new features

- **AI text reflow** — adjusts copy to fit frame without manual editing
- **Variable font axis UI** — per-paragraph optical size, weight and width axis
- **Multi-channel publish panel** — one click to print PDF, EPUB and web simultaneously
- **Content Credentials** — embeds attribution metadata on export
- **Generative layout** — AI suggests frame arrangement from content brief
